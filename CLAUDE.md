# CLAUDE.md

本仓库为 Windows Electron Markdown 阅读器（MarkdownReader）。开发与改动时请同时阅读 `README.md` 中的产品边界与限制。

## 与用户交互（提问）

当需要用户在一组明确选项中做选择（例如是否调研、是否写入文档、多分支方案取舍）时，应使用 **AskQuestion** 工具呈现题目与选项，而不是仅在正文中罗列 A/B/C 让用户打字回复。若选项无法穷举或需要开放叙述，再在聊天中补充说明。

## Design System

始终在执行任何视觉或 UI 相关改动前阅读根目录的 `DESIGN.md`。字体、颜色、间距、圆角、动效与美学方向以该文档为准；未经用户明确同意不要偏离。进行 QA 或界面审查时，对不符合 `DESIGN.md` 的实现应主动标出。
